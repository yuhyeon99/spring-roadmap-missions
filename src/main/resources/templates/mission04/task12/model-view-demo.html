Model에 담아 전달한 핵심 데이터

이 영역은 컨트롤러가 `Model`에 넣은 `session`, `coachMessage`, `preparedDate`, `keyPoints` 값을 템플릿이 읽어 화면으로 표현한 결과입니다.

컨트롤러는 HTML 태그를 직접 만들지 않고, 어떤 데이터를 보여줄지만 결정합니다. 실제 문단, 카드, 목록 배치는 템플릿 파일이 담당합니다.
